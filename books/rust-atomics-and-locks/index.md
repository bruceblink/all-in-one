# Rust Atomics and Locks 中文翻译

## 简介

《Rust Atomics and Locks》by Mara Bos 是一本围绕 Rust 并发底层原语（atomics、锁、内存排序、操作系统支撑等） 的实用书籍。
总体来说，它不是一本入门 Rust 的书，而是 专门针对并发和底层细节 的深度资料

## 目标读者

- 对并发和底层原理感兴趣的 Rust 程序员
- 对跨语言并发机制感兴趣的工程师
- 已经具备基本 Rust 并发经验的进阶读者
